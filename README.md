# TaskFlow

TaskFlow is a Flutter application for personal task management.

The project is being developed as a portfolio application following modern software engineering practices, including:

- Clean Architecture
- Feature-first project organization
- Firebase integration
- Riverpod state management
- GitHub Actions CI
- Pull Request workflow
- Architecture Decision Records (ADR)

---

## Features

Current roadmap:

- User authentication
- Task management
- Categories
- Priorities
- Due dates
- User profile
- Settings

## Getting Started

Clone the repository:

```bash
git clone https://github.com/frozzen-ex/taskflow.git
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

Run tests:

```bash
flutter test
```

Analyze the project:

```bash
flutter analyze
```
